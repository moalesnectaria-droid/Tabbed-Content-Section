# Tabbed Content Section

A simple responsive Tabbed Content Section built using HTML, CSS, and JavaScript.

## Features
- Multiple tabs (Overview, Skills, Contact)
- Dynamic content switching with JavaScript
- Active tab highlighting
- Clean and responsive design
- Smooth transitions

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
